## 📌 Project Overview

**JobMatch ET** is an **AI-powered recruitment platform** specifically designed for the Ethiopian job market. It uses **Machine Learning (Random Forest)** to match job seekers with employers based on a 4-factor scoring system.

### 🎯 Key Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI-Powered Matching** | Random Forest ML with 78.5% accuracy |
| 📊 **4-Factor Scoring** | Skills (40%) + Experience (25%) + NLP (20%) + Education (15%) |
| 👨‍💼 **Candidate Portal** | Profile management, skill suggestions, apply to jobs |
| 🏢 **Employer Portal** | Post jobs, rank candidates, manage applications |
| 👑 **Admin Dashboard** | User management, job management, system analytics |
| 📄 **CV Upload** | Upload and share CV with employers |
| 🎓 **Education Tracking** | GPA, specialization, field of study |
| 🔬 **NLP Analysis** | Semantic matching of job descriptions |
